室蘭SUPER BASH

北海道（北海道地方）

開催頻度

次回開催日
2026-06-20
会場
中央埠頭特設会場

北海道室蘭市海岸町の中央ふ頭特設会場で開かれる野外音楽フェスです。 club RED HILLが主催し、室蘭市などが後援する、道内最大規模の野外レゲエイベントとして知られています。 レゲエやヒップホップを軸に毎回約25組のアーティストが港まちに集結し、全国から訪れる約800人のファンと共に室蘭港に熱い歌声を響かせます。 会場にはキッチンカーなど多数の店舗も並びます。 2026年は6月に開催されます。

出演（2026年）: PAPA B / VIGORMAN / 寿君 / RAY / 775 / RAFUU / ST fr.HACNAMATADA / LIFE STYLE / 柊人 / NG HEAD / EXPRESS / BUDDHA MAFIA / 妖艶金魚 / GREEN PEACE / WORM HOLE / WACCHI / MARY kidz wiz YURA / MACKIE祭SQUAD / CHOP STICK
