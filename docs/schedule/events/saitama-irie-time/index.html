IRIE TIME

埼玉県（関東地方）

会場

埼玉県羽生市の「きやっせ羽生」で開催される、入場無料の音楽フェスティバル。 地元を拠点とするプロジェクトクルーが立ち上げた取り組みで、公式は「音楽を通して、人と人、人と街がつながるきっかけを作りたい」というコンセプトを掲げます。 レゲエやサウンドシステムを中心としたライブステージとDJパフォーマンスに、キッチンカーや飲食ブース、交流スペースを併設。 2026年7月の開催で、街と音楽がゆるやかにつながる一日が広がります。

出演（2026年）: 【7/25】YUTEN / OGA Jah Works / Masazaburro / MAXIMUM / DEAL ROCK SOUND / Shoma fr.dambosound / DJ WATA / LION POWER SOUND
