BONJOVI盆踊り

山梨県（中部地方）

開催頻度
6月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）
次回開催日
2026-06-06
会場
都留文科大学駐車場

山梨県都留市の都留文科大学駐車場で開催される、Bon Jovi をもじったネーミングが目を引く野外音楽イベントです。 公式は「Bon Jovi盆踊り」を掲げ、都留市での初開催を打ち出しています。 入場無料でレゲエ寄りのラインナップを軸に、DOZAN11 aka 三木道三、CHEHON ら 18 組超が出演し、子どもも楽しめる屋外フェスとして 2026 年 6 月開催を予定しています。

出演（2026年）: DOZAN11 aka 三木道三 / CHEHON / EXPRESS / SOY fr humancrest / JUN fr BANTY FOOT / MjakiTheClan / Ragga8 / ANNA / DJ MIYUKI / DJ RINKA / SASSYS BASE / DJ CKS / WATTZ / Generation5 / 富士学苑中学高等学校ジャズバンド部 ムーンインレットサウンズオーケストラ / SuperSquad / KUSURI / avex dance master
