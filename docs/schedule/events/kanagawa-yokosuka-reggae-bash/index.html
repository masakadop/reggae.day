YOKOSUKA REGGAE BASH

神奈川県（関東地方）

9月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）

2026-09-12
会場

神奈川県横須賀市で開催される野外レゲエフェスティバルです。 公式は「未来への種蒔き」を掲げ、愛すべき地元横須賀から多くの人へ、きっかけという名の種を届けることをコンセプトに据えています。 1秒先からまだ見ぬ未来へと繋がる思いを音楽で共有する場として続き、レゲエを軸に多彩なアーティストが集う地元発の野外音楽フェスです。 2026年は9月の開催が公式より告知されています。 YOKOSUKA REGGAE BASH 2023 LAUNCH MOVIE
