福島県（東北地方）

「新時代」は、福島県郡山市のビッグパレットふくしまを会場に開催されるミュージックフェスティバルです。 公式は「音楽は、ただの娯楽ではない。 心を震わせ、背中を押し、時代を動かす力」と掲げ、若者の内に眠る衝動と可能性を解き放つ場であることを標榜します。 夢を動かし始める場所として、ライブアーティストにダンス、サウンド／DJなど多彩な要素を組み合わせ、2026年7月20日の海の日に開催されます。

出演（2026年）: 柊人 / DOZAN11 aka 三木道三 / EXPRESS / BANTY FOOT / KEN-U / CHEHON / JUMBOY / Spinna B-ILL / RYO the SKYWALKER / KOJOE / ZENDAMAN / オヤカタくん / BLACKANESE / CHOP STICK / SUGAI QUINTET + MICHIYO / Dance Studio Morely / STUDIO SHEIL / OLD RIVER / MAD FIXXER / Classic Lovers Club / BrAt Dance School / 7color dance studio / S,S,S Dance Studio / Dance Studio Righggle / SONIC BOOM / MINORITY SOUND / RISING SOUND / BLACK BENZ SOUND / SOUD SYSTEM / Shing Ji Dai / HOMETOWN Hi-Fi
