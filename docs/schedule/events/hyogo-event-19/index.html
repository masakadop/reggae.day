播州ごじゃフェス

兵庫県（近畿地方）

開催頻度
9月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）
次回開催日
2026-09-22
会場
マリンリゾート姫路白浜

兵庫県姫路市の海辺、マリンリゾート姫路白浜を舞台に開かれる音楽の祭りです。 公式は「播州恒例この祭」「今年もこじゃアツくなる」と掲げ、播州の地で続く恒例イベントとして発信しています。 サウンドシステムを据えたステージやフードエリアが設けられ、音楽とダンス、そして海辺ならではの開放感を一体で味わえるのが持ち味です。 2026年は9月22日に開催され、出演者などの詳細は近日公開予定とされています。
