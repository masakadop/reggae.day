http://www.omofes.jp 秋田県横手市雄物川町の雄物川中央公園で開催される、ジャマイカン・ミュージック・フェスティバル「OMOFES（おもフェス）」です。 公式は「SKA／REGGAEを生みだしたジャマイカとほぼ同じ面積の秋田県で本場ジャマイカの古き良き空気感を再現する」と掲げ、地元・東北・東京のアーティストが集います。 屋外ステージで開催され、キッチンカーや屋台・マルシェも併設、入場無料で楽しめる地域密着型のフェスです。 2026年は7月20日に開催されます。
