Splash Mountain

栃木県（関東地方）

開催頻度
9月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）
次回開催日
2026-09-06
会場
岩船山採石場跡地
イベント公式サイト
https://splashmountain.vip

栃木県栃木市の岩船山採石場跡地で開催される野外フェスです。 公式は、大迫力の岩山をバックにしたメインステージにレゲエアーティストをメインとした個性的な顔ぶれが集うと掲げています。 サブステージには宇都宮のレゲエイベント「MIYAMIYA Saturday」とのコラボエリアを設け、B級グルメの飲食ブースや家族でも安心のテントブースも用意。 夜は岩山と打ち上げ花火が彩る、夏の終わりにふさわしい一日です。 2026年9月6日開催。

出演（2026年）: 【9/6】CHOP STICK / KENTY GROSS
