姫路お城まつり サウンドフェスティバル

兵庫県（近畿地方）

2026-05-24
会場

姫路お城まつりサウンドフェスティバルは、戦後復興のシンボルとして1948年に第1回が開催された「姫路お城まつり」の一環として行われる音楽イベントです。 会場は姫路市本町のシロトピア記念公園野外ステージ。 2026年は5月24日(日)11時から17時まで開催され、今年でvol.48を数える長く続くプログラムです。 姫路お城まつり全体は2026年5月22日から24日までの3日間にわたって行われます。
