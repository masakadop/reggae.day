会場
しまなみビーチ

広島県尾道市・因島の「しまなみビーチ」を舞台に、夏のビーチを盛り上げる2日間の音楽＆ウェイクボードイベントです。 全国各地から選手が集うウェイクボード大会と、ライブパフォーマンス、ダンス、飲食ブースが融合し、海辺ならではの開放的な雰囲気のなかで楽しめます。 イベントエリアの入場料は無料に設定されており、気軽に立ち寄れる構成です。 2026年は7月19日（日）・20日（月・祝）の2日間にわたり開催されます。
