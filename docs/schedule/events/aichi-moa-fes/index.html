MOA FES

愛知県（中部地方）

開催頻度
9月頃開催（前年に続く開催。今後の定期開催有無は公式サイトでご確認ください）
次回開催日
2026-09-27
会場
Comas hui Beach park
イベント公式サイト
https://moa-fes.com/

南知多の海辺・Comas hui Beach parkで開かれる音楽フェスです。 昨年に続く開催となる今回は、サンセットを眺めながら音楽に身を委ね、広大な海を感じられる一日を掲げています。 フードやドリンク、子ども向けのアミューズメントも揃い、ペット同伴も可能。 大人から子どもまで自然と笑顔になれる場所を目指す、家族や仲間で一日中楽しめる空間です。 2026年9月開催。

出演（2026年）: 寿君 / HISATOMI / RIVA / ちーかまん / BLACK BLAZE
