REGGAE SUPER POWER キセキ

東京都（関東地方）

開催頻度
8月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）
次回開催日
2026-08-02
会場
吉祥寺CLUB SEATA

東京都武蔵野市の吉祥寺CLUB SEATAを舞台に開催される、レゲエカルチャーをテーマに掲げる音楽イベントです。 公式は「キセキ」をサブタイトルに冠し、これまで複数回の開催を積み重ねてきた歴史を持ちます。 会場はライブハウス型の屋内ステージで、ゲストアーティストとゲストサウンドが一堂に集う構成となっており、レゲエの一夜を作り上げます。 2026年は8月2日に単日で開催される予定です。 REGGAE SUPER POWER -キセキ2023- 【Thanks Movie】

出演（2026年）: EXPRESS / TAK-Z / ARARE / BANTY FOOT / J-REXXX / NG HEAD / 導楽 / RAY / BES
