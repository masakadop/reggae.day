会場
マゼノ共和国 甲の瀬キャンプ村

「DANCEHALL JOURNEY in 阿蘇」は、熊本県阿蘇郡南小国町のマゼノ共和国甲の瀬キャンプ場を舞台に、夕方から翌朝までぶっ通しで繰り広げられる野外レゲエフェスです。 公式は「九州で唯一のオールナイト野外レゲエフェス」を掲げ、キャンプをしながらレゲエに身を委ね、キャンプファイヤーを囲んで家族や仲間と夏の思い出を作る滞在型イベントとして位置付けています。 2026年は8月22日に開催されます。 LIVEダイジェスト2 DANCEHALL JOURNEY in 阿蘇2025
