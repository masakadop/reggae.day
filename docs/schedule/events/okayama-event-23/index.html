玉野レゲエ祭は、岡山県玉野市の宇野港第一突堤を会場に開かれる、長く続く屋外レゲエフェスティバルです。 瀬戸内の港を舞台に、ライブショーケースやサウンドタイムを軸としたレゲエサウンドが一日を彩ります。 会場ではX-SPORTS体験やフード、マーケット、カーブースも展開され、音楽と食、遊びを一度に楽しめる構成が特徴です。 15歳以下は入場無料で、家族連れでも気軽に足を運べます。 2026年は8月に開催されます。
