みど村レゲエ祭は、埼玉県秩父郡小鹿野町のみどりの村特設ステージで毎年ゴールデンウィークに開催されている野外レゲエフェスです。 公式は「大自然を感じながら音楽に浸れる」場として打ち出し、シャトルバスの運行や隣接キャンプ場との宿泊連携など、長時間ゆったり過ごせる設計が特徴。 2026年5月4日にはCHEHONや寿君、RYO the SKYWALKERらが出演します。 祝！10周年！みど村レゲエ祭

出演（2026年）: 【5/4】CHEHON / 寿君 / RYO the SKYWALKER / CHOZEN LEE / 775 / RAY / TAKAFIN / 導楽 / GUAN CHAI / POWER WAVE / SHUN / TAKE-ROLL / 番頭 / SHION
