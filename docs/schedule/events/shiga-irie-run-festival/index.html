滋賀作 – Irie Run Festival

滋賀県（近畿地方）

開催頻度
7月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）
次回開催日
2026-07-19
会場
高島びわっこパーク

『滋賀作 – Irie Run Festival』は、琵琶湖の湖畔を舞台に毎夏開催されるレゲエフェスティバルです。 会場は滋賀県高島市の高島びわっこパークで、湖と空が広がるビーチサイドのロケーションが魅力。 CHEHONやNG HEAD、HISATOMIら国内レゲエシーンを牽引する顔ぶれに加え、各地のサウンドクルーやダンスショーも集結し、アイリーな一日を彩ります。 2026年7月開催。

出演（2026年）: 【7/19】CHEHON / NG HEAD / 韻踏合組合 / EXPRESS / 寿君 / 775 / HISATOMI / G.B.C CAMP / PETER MAN / MASAZABURRO / Ryu-Z / Yoppy / 空助 / SUNS / T-SLUGGER / SURPRISE / KING LYCHEE / BURN DOWN / KING JAM / BANTY FOOT / ARTICAL KING / ONLY SOUND / SHAKEY’S / GREEN WAVE / DJ YOSHI / KING SONIC / YAH-MAN☆ / BACK UP DANCER RENA / VOICE OF THE LAKE / JUNYA “Thirdeye” / S-STEDY / TIGER WORKS / CHOP STICK / MANATY
