石川県発の地域団体『ONE PEACE🤝ISHIKAWA』が主催する音楽イベントです。 公式は『石川県から始まる新しい波』を掲げ、支援団体ではなく県民が横で繋がる場として、その輪の力で能登の子どもたちに笑顔を届ける企画を打ち出しています。 会場は能登半島・志賀町の道の駅とぎ海街道周辺多目的広場。 2026年6月28日に開催され、ライブやDJのパフォーマンスが繰り広げられます。 能登の子供達を笑顔にする大作戦
