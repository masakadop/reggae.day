YACHIROCK MOVEMENT

千葉県（関東地方）

YACHIROCK — 公式サイト / SNS / SNS

千葉県八千代市発の野外音楽フェスです。 地元でのゴミ拾いや募金活動を母体に生まれ、2023年に初開催したやちロックが、2026年5月『YACHIROCK MOVEMENT2』としてスケールアップして帰ってきます。 会場は竹林の美しい景観を活かしたキャンプ場「かぐや」。 竹に囲まれたプライベート感覚のサイトで、地元農家の新鮮な野菜も味わえます。 音楽と笑顔が混ざり合う、子どもも大人も楽しめる居心地のよい一日を届けます。

出演（2026年）: 【5/23】韻シスト BAND / CHOZEN LEE & THE BANG ATTACK / 鎮座 DOPENESS / SATTU CREW / BIG MOUTH BRASS BAND【5/24】光風 &GREEN MASSIVE / 三宅洋平 / ROWHOO / BIG MOUTH BRASS BAND
