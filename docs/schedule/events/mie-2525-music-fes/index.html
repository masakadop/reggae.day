『2525 MUSIC FES』は、三重県松阪市の中部台運動公園を舞台に、移動式マルシェ「2525marché」が「A FUN TIME FULL OF SMILES」を掲げて2025年に立ち上げた音楽フェスです。 レゲエ・ヒップホップ・R&Bを軸にVIGORMANらが出演する有料ライブエリアと、約100店舗のフード・マーケットやキッズエリアが楽しめる無料エリアが共存。 2026年5月に2回目を迎えます。

出演（2026年）: VIGORMAN / SPICY CHOCOLATE / CHICO CARLITO / EXPRESS / KOTA / lonelow / 韻マン / 竹内唯人 / 當山みれい / 絹
