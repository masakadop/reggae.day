AWAJI PARKING FESTIVAL

兵庫県（近畿地方）

4月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）

会場
多賀の浜海水浴場
イベント公式サイト

AWAJI PARKING FESTIVAL vol.3 は、開放的な淡路島の海辺から人々を楽園へと導くカルチャーイベントです。 会場は兵庫県淡路市の多賀の浜海水浴場で、音楽LIVE、DJ、フード、アートが交差する一日限りの催しとなっています。 Special Live には日本屈指のレゲエバンド YOUTH OF ROOTS が参加し、忙しない日々から抜け出してゆったりとリラックスできる時間を演出します。 2026年4月開催予定。

出演（2026年）: youth of roots / bbb / dj s.o.l / 91wine / miyacii / nakarchy / onion vibration
