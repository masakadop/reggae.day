肉レゲエフェスティバル〜加古川〜

兵庫県（近畿地方）

開催頻度
9月頃開催（2026年の開催情報です。定期開催かどうかは公式サイトでご確認ください）
次回開催日
2026-09-22
会場
加古川河川敷 どうぶつの森広場

「肉レゲエフェスティバル〜加古川〜」は、音楽エンターテイメントと肉料理を中心とした出店のコラボレーションを掲げる、兵庫県加古川市のレゲエフェスティバルです。 会場は加古川河川敷のどうぶつの森広場で、市内のお肉屋さんを中心とした飲食ブースとレゲエアーティストによるステージが楽しめるイベントとして公式が紹介しています。 2026年は9月22日に開催が予定されています。
