五龍王祭 DEAL ROCK

栃木県（関東地方）

会場

「五龍王祭 DEAL ROCK」は、栃木県日光市の日光国立公園 龍王峡 特設会場で開かれる入場無料の野外音楽フェスティバルです。 公式は「DEAL ROCK with SOUND SYSTEM」を冠し、サウンドシステム・スタイルを軸とした一日のプログラムが組まれます。 日光市および日光市観光協会鬼怒川・川治支部の後援を受け、2026年は5月17日に開催されます。

出演（2026年）: DOZAN11 aka 三木道三 / PAPA-B / J-REXXX / 導楽 / CHOP STICK / プリンスヌマニエル / ICHINO / JUNKY’S / TAKE ROLL / NOBU / CHANCO MELODY / EMICA / PRAYING HANDS
