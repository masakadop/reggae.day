SAVE The TSUYAMA STREET

岡山県（中国地方）

会場
グリーンヒルズ津山 野外ステージ

「SAVE The TSUYAMA STREET」は、岡山県津山市のグリーンヒルズ津山・野外ステージで毎年開催される入場無料のチャリティーイベントです。 市街地を見渡す芝生の丘を舞台に、ヒップホップやレゲエなどを軸とした地元アーティストのライブやダンスチームのパフォーマンス、グルメブースが並び、子どもから大人まで気軽に集える地域密着の一日を作り上げます。 2026年8月15日に開催予定です。
