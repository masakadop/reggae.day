「ソトヂカラ」は、滋賀県大津市・琵琶湖畔の近江舞子南浜特設会場で開かれる野外レゲエフェスです。 アーティストG-MANをナビゲーターに、日本のレゲエ／ダブシーンを担うアーティストとサウンドクルーが集う、湖畔のビーチを舞台にしたアーティスト主催型のイベント。 2026年は5月31日(日)、開場12時から終演20時まで、サウンドシステムカルチャーを存分に体感できます。
