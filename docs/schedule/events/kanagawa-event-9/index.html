神奈川県（関東地方）

主催

『渋谷レゲエ祭』は、SPICY CHOCOLATEが2009年からスタートしたレゲエフェスで、公式によれば2026年で19回目を迎えます。 会場は神奈川・川崎のCLUB CITTA’で、2026年10月4日に開催。 HOST SOUNDをSPICY CHOCOLATEが務め、CHEHONや寿君らレゲエシーンのアーティストが集う、ダンスホールの熱気にあふれた一日が予定されています。 SPICY CHOCOLATE presents「渋谷レゲエ祭2025」After Movie

出演（2026年）: SPICY CHOCOLATE / CHEHON / EXPRESS / 寿君 / RAY / REKID
