http://www.highestmountain.jp 「ハイエストマウンテン」は、大阪府豊中市の服部緑地野外音楽堂を舞台に開かれる、屋外型のレゲエ・フェスティバルです。 公式は「MIGHTY JAM ROCK presents」を掲げ、ジャパニーズ・レゲエシーンを牽引する顔ぶれが2デイズ編成で集い、公式が発表したラインナップが土日にわたって披露されます。 2026年は8月22日（土）・23日（日）の2日間にわたって開催されます。
