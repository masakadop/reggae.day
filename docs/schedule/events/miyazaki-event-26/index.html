宮崎シーサイドフェスは、宮崎市青島のAOSHIMA BEACH PARKで開かれる入場無料の海辺の音楽イベントです。 公式は「こどもたちを海で遊ばせ、海難事故を撲滅する」を掲げ、マリンアクティビティと海の安全ワークショップに、DJタイムと夕暮れのサンセットライブを組み合わせた『遊び×学び』のエデュテインメント型フェスとして実行委員会が運営します。 2026年9月開催。

出演（2026年）: DOZAN11 (三木道三) / SHADY
